# Respostas


## #1 Questão

  


Qual será a saída (resultado) para os valores de p = 1 e q = 0 dentro da tabela-verdade AND?

  

- [ ] Retorna verdadeiro (1). 

- [ ] Não pode retornar valor nenhum.

- [x] Retorna falso (0). ✅

- [ ] Retorna uma negação - !p.

- [ ] Retorna uma negação - !q.
